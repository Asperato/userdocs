asperato ONE

Documentation

Asperato ONE is an AppExchange package which allows you to control payment collection from Salesforce in a secure, audited and compliant fashion. We support multiple payment methods (such as card, direct debit, eCheck, Paypal, etc.) and any payment processor, so long as they have a compatible API.

Haven't yet got the package? Install it from the AppExchange here.

The documentation here will walk you through installing, setting up, using and customising the package. If you have any questions not covered here, please email support@asperato.com and we'll do our best to help.
